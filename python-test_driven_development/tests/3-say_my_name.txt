==================================
How to use 2-matrix_divided module
==================================

Importing the function:
    >>> matrix_divided = __import__('2-matrix_divided').matrix_divided

Normal case:
    >>> matrix = [[1, 2, 3], [4, 5, 6]]
    >>> matrix_divided(matrix, 3)
    [[0.33, 0.67, 1.0], [1.33, 1.67, 2.0]]

Division by float:
    >>> matrix_divided([[10, 20], [30, 40]], 2.5)
    [[4.0, 8.0], [12.0, 16.0]]

Error: div is zero:
    >>> matrix_divided([[1, 2], [3, 4]], 0)
    Traceback (most recent call last):
        ...
    ZeroDivisionError: division by zero

Error: div is not a number:
    >>> matrix_divided([[1, 2], [3, 4]], "three")
    Traceback (most recent call last):
        ...
    TypeError: div must be a number

Error: different row sizes:
    >>> matrix_divided([[1, 2, 3], [4, 5]], 2)
    Traceback (most recent call last):
        ...
    TypeError: Each row of the matrix must have the same size

Error: matrix elements not numbers:
    >>> matrix_divided([[1, "two"], [3, 4]], 2)
    Traceback (most recent call last):
        ...
    TypeError: matrix must be a matrix (list of lists) of integers/floats
